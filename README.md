## Installation

Follow the steps below for first time installation:

`npm init`

`npm i bcrypt`

`npm i connect-session-sequelize`

`npm i dotenv`

`npm i express`

`npm i express-handlebars`

`npm i express-session`

`npm i mysql12`

`npm i sequelize`

`npm i spotify-web-api-node`
